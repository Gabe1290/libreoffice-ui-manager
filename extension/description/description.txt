LibreOffice UI Manager (LOUIM) helps teachers progressively simplify the
LibreOffice Writer, Calc, Impress, and Draw interface for learners. Apply a .louim template to hide or
show menus, individual menu items, toolbars, toolbar buttons, sidebar decks, and
extension menus, and restore the full interface at any time. Save your own
layout as a template to share a "beginner" profile across machines. Available in
English, French, German, and Italian.
