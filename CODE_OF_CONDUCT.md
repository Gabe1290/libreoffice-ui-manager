# Code of Conduct

LOUIM is built by and for educators. Be kind, be patient — especially with
first-time contributors and non-native English speakers — and assume good
faith. Harassment, personal attacks, and discriminatory language are not
tolerated anywhere in the project's spaces.

This project follows the spirit of the
[Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).
Report unacceptable behavior privately to the maintainer via GitHub.
