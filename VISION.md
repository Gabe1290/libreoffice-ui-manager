# LibreOffice UI Manager — Vision

LibreOffice UI Manager (LOUIM) helps teachers and learners use LibreOffice progressively.

Its goal is simple:

> The interface adapts to the learner, not the learner to the interface.

Many students and adult beginners are overwhelmed by the full LibreOffice interface. LOUIM allows teachers to reveal only the tools needed for the current learning objective, then progressively introduce more features.

LOUIM is not a lockdown tool. It is a learning tool.

## Long-term vision

In the long term, LOUIM should become a cross-platform educational extension for LibreOffice that supports:

- Writer
- Calc
- Impress
- Draw

Teachers should be able to create and share simple templates that configure the interface for a lesson, a learning stage, or a specific classroom context.

## First goal

The first working version focuses only on LibreOffice Writer.

It should allow users to:

- import a Writer interface template
- export a Writer interface template
- export the current Writer interface as a template
- apply a template
- restore the full Writer interface

## Guiding motto

Simplify first. Reveal progressively.
